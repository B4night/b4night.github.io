hexo+github建github.io

安装hexo+连接github, 请看pdf, 来自B站教程

以下是如何将joplin文档通过joplin-blog这个插件发布

  1. npm install -g joplin-blog, 安装joplin-blog
  2. 在hexo文件根目录下创建.joplin-blog.json, 并配置. 配置请看github
  3. 在当前目录下执行npx joplin-blog export, 自动将配置文件中对应标签的文档载入到/source/_posts/文件夹中, 之后这些文档会被上传
  4. hexo g & hexo d. 自动上传到github

可以通过一个脚本来执行
